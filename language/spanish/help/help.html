gwloto

Visión de conjunto

gwloto, geekwright lockout tagout, es una energía peligrosa en Internet Control Manager plan. Está diseñado para facilitar la comunicación y el seguimiento de los procedimientos de seguridad vitales para que los trabajadores de seguridad realizar tareas de mantenimiento a través de un bloqueo y etiquetado de seguridad documentado programa.

gwloto se implementa como un módulo para sistemas de XOOPS. Esto le da al integrador de sistemas una gran flexibilidad en la implementación de una solución de etiquetado de bloqueo, que ofrece muchas posibilidades, de un sistema gwloto autónomo a un portal de seguridad integral.

Uso

gwloto es una herramienta flexible y potente, con muchas opciones. completar documentación y una demo están disponibles en gwloto.geekwright.com.

TCPDF

TCPDF es una clase PHP para generar documentos en formato PDF, y se requiere para la mayoría de plugins se incluyen para operar. Para obtener más información sobre TCPDF consulte www.tcpdf.org.

TCPDF se detectará automáticamente si se instala en el directorio gwloto en una carpeta llamada TCPDF. La ubicación también se puede especificar en las preferencias del módulo gwloto.
