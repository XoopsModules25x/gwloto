gwloto

Overview

gwloto, geekwright lockout tagout, is a web based hazardous energy control plan manager. It is designed to facilitate the communication and tracking of vital safety procedures to enable workers to safely accomplish maintenance tasks through a documented lockout and tagout program.

gwloto is implemented as a module for XOOPS systems. This gives the system integrator great flexibility in implementing a lockout tagout solution, providing many options, from a standalone gwloto system to a comprehensive safety portal.

Usage

gwloto is a flexible and powerful tool with many options. Complete documentation and a demo are available at gwloto.geekwright.com.

TCPDF

TCPDF is a PHP class for generating PDF documents, and is is required for most supplied plugins to operate. For more information on TCPDF please see www.tcpdf.org.

TCPDF will be automatically detected if installed in the gwloto directory in a folder named tcpdf. The location can also be specified in the gwloto module preferences.
